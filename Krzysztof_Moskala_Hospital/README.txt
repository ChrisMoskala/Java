To run the program simply run the executable file or if you have a java installed .jar
file directly. Once you log in, you can experiment with the functions and relations between
classes, mostly focused on staff, patient, pharmacy and equipment. Files are set to be
saving on the desktop - 11 files in total for saving, same ammount for backups with
specified names. If something is unclear, you can always click the check button,
most of the information about an object is stored in a toString of each class.
The array size is set to be for 10000 people for both staff and patients.
